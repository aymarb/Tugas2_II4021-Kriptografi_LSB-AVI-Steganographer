# Tugas2_II4021-Kriptografi_LSB-AVI-Steganographer
Steganografi LSB pada Berkas Video AVI.

Aplikasi steganografi berbasis GUI untuk menyisipkan dan mengekstraksi pesan rahasia pada berkas video berformat AVI menggunakan metode Least Significant Bit (LSB). Program ini dapat menyisipkan pesan berupa teks maupun berkas. Fitur utama program mencakup opsi enkripsi pesan dengan algoritma A5/1, mode penyisipan sekuensial maupun acak menggunakan stego-key , serta fitur analisis kualitas stego-video melalui perhitungan Mean Squared Error (MSE), Peak Signal-to-Noise Ratio (PSNR), dan visualisasi histogram warna RGB.

---

## Teknologi yang Digunakan

- **Bahasa Pemrograman:** Python 3
- **Antarmuka Pengguna (GUI):** CustomTkinter
- **Pemrosesan Citra & Video:** OpenCV (`cv2`), NumPy
- **Visualisasi Data:** Matplotlib
